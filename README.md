## Sobre o Projeto

O **MonitoreSeuTreino** permite ao usuário:

- Criar e editar fichas de treino semanais
- Registrar a execução de sessões de treino (séries, repetições, carga)
- Consultar o histórico de treinos realizados
- Visualizar um resumo semanal de constância
- Receber sugestões de treino com base no perfil (onboarding)
