## Introdução

Este módulo apresenta a modelagem BPMN (Business Process Model and Notation) do projeto G7_MonitoreSeuTreino. O objetivo é representar visualmente os processos metodológicos adotados pela equipe durante o desenvolvimento, incluindo o fluxo de trabalho, as cerimônias e as práticas de engenharia de software escolhidas.

A modelagem BPMN foi definida como tarefa na [Ata de Reunião 01](Base/1.5.IniciativasExtras.md), sob responsabilidade de Daniel Teles, e tem como finalidade formalizar o processo de desenvolvimento para que todos os membros tenham clareza sobre papéis, atividades e fluxos de decisão.
