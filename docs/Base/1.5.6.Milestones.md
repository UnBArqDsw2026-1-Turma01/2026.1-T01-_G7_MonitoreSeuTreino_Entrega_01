# 1.5.6. Milestones do Projeto

## Introdução

As milestones organizam o desenvolvimento do MonitoreSeuTreino em fases progressivas do MVP, partindo das funcionalidades essenciais até as complementares. A distribuição das 29 user stories foi baseada no **índice numérico de priorização** (Impacto ÷ Complexidade), calculado via Planning Poker, buscando um balanceamento equilibrado entre as fases (~10 itens cada).

[Visualizar milestones no GitHub](https://github.com/UnBArqDsw2026-1-Turma01/2026.1-T01-_G7_MonteSeuTreino_Entrega_01/milestones ':target=_blank')
